# Cockpitdecks Extension for X-Plane Flight Simulator software

This extension allows Cockpitdecks to connect to Laminar X-Plane Flight Simulator.

## X-Plane plugins

XPPython3 plugins for X-Plane (e.g. `PI_CockpitdecksFMS.py`) have moved to the [cockpitdecks-xplane-plugins](https://github.com/dlicudi/cockpitdecks-xplane-plugins) repository.
